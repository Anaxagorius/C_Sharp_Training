
# Member Registration (WPF MVP)

## IMPORTANT
This is a **WPF** project (not WinForms). If you try to paste these files into a WinForms project,
you will see errors such as:
- MessageBoxButton / MessageBoxImage not found
- System.Windows.Controls missing

## Run
1. Open MemberRegistrationMVP.sln in Visual Studio 2022+
2. Press F5

## Data file
- MemberMaster.txt is created next to the EXE (bin/Debug/...)

## Features
- Register (Add) auto-saves
- Load / Save buttons
- Select item -> read-only + prompt to edit + Update/Delete shown
- Delete confirmation
